# Migration-path disposition

`DevelopmentCommonAuthorityTrial` remains repaired but development-only. The StandardIo exception was caused by treating an absolute constraint coordinate as a non-negative magnitude; negative canvas coordinates are now accepted and covered by a regression test.

The trial is historical comparison/evidence generation. It is not the production migration route and is not used by `GenerateProjectRegion`.

Retained migration-only areas and deletion gates:

| Area | Status | Deletion gate |
|---|---|---|
| component alternative search | frozen | delete when complete project regions cover its supported families and no active evidence report consumes it |
| mixed positional movement closure | frozen | delete when placement is owned per attempted project/span |
| link-level component projector/classifier | frozen | delete when fallback is exclusively project/span granular |
| opposing-scope search | frozen | delete with the alternative component search |
| persistent mixed constraints and revision/invalidation | retained for trial only | delete after immutable project generation covers movement-required fixtures |
| legacy path parity/reconstruction | retained for historical trial | delete after the project fixture catalogue replaces parity evidence |
| mixed rollback and attribution | retained for historical trial | delete when no production candidate is validated inside a mixed legacy graph |
| `LegacyRoutingPipeline` | normal production | delete incrementally only for approved complete-region selectors |

Reusable target primitives retained: positional hierarchy, subtree envelopes, natural sizing, terminal allocation, inter-layer discovery, deterministic slots, vertical/return columns, traceability validation, ownership segmentation, project bounds, and serialization.
